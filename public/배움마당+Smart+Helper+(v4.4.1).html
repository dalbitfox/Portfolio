🎓 배움마당 Smart Helper (v4.4.1)
위 보라색 버튼을 마우스로 누른 채, 브라우저 북마크(즐겨찾기) 바로 끌어다 놓으세요!
* 북마크 바가 안 보인다면 키보드의 Ctrl + Shift + B를 누르세요.
1
LGU+ 배움마당 사이트에 로그인한 뒤 수강 대상 과목 상세 화면(차수별 학습하기가 나열된 곳)으로 이동합니다.
2
등록해 둔 즐겨찾기 바의 [🎓 배움마당 도우미] 북마크를 마우스로 가볍게 클릭합니다.
3
화면에 즉시 스마트 대시보드가 로드되며, 수강율 100% 미만인 강의들을 대상으로 백그라운드 동시 수강을 진행합니다!
💡 드래그가 잘 되지 않거나 동료에게 메신저로 전달할 때:
복사 후, 북마크 관리자나 아무 사이트나 북마크를 만드신 뒤 [주소(URL)] 입력칸에 복사한 코드를 그대로 붙여넣고 저장해 쓰시면 됩니다.
🔄 [시험 자동 풀이 무한 지속] 100% 전자동 연속 풀이 가이드 (권장) 영속화 솔루션
일반 즐겨찾기(북마크클릿) 방식은 브라우저 보안 규격상 페이지가 새로고침되거나 이동할 때 도우미 작동이 초기화됩니다.
아래 Tampermonkey(템퍼몽키) 방식을 이용하시면, 시험 페이지가 넘어가도 어떠한 조작이나 재클릭 없이 AI가 다음 문제를 연속해서 완전 자동으로 풀이합니다.
1
브라우저에 맞는 마켓에서 Tampermonkey 확장프로그램 (클릭하여 설치)을 설치합니다.
2
아래 [유저스크립트 코드 복사] 버튼을 클릭합니다.
3
브라우저 우측 상단의 Tampermonkey 아이콘 클릭 ➔ "새 스크립트 만들기..."를 선택합니다.
기존에 적힌 템플릿 텍스트를 전부 지운 뒤, 복사한 코드를 그대로 붙여넣고 저장(Ctrl + S)하면 즉시 영구 적용 완료!
